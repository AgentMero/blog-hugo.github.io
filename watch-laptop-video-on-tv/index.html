Watch Laptop Videos on TV

Turn On SMB(Server Message Block)

Window Data Sharing Protocol

  • Go to Control Panel
  • Go to Programs and Features
  • Click the Turn Windows features on or off
  • Check SMB 1.0/CIFS File Sharing Support -> SMB 1.0/CIFS Client and SMB 1.0/CIFS Server

Add Guest Account

  • Go to Start -> Setting -> Accounts
  • Go to Family & other users
  • Click Add someone else to this PC
  • Select I don't have this person's sign-in information, and on the next page, select Add a user without a Microsoft account.
  • Enter a user name, password, or password hint—or choose security questions—and then select Next.

Shared Folder

  • Right Click the target-shared-folder
  • Go to Properties
  • Go to Sharing
  • Click Share
  • Add Guest
  • Change the right to Read/Write
  • Click Advanced Sharing
  • Check Share this folder

Network Setting

  • Go to Control Panel
  • Go to Network and Sharing Center
  • Click Advanced sharing settings
  • Go to Private and turn on Network Discovery and File and printer sharing
  • Go to Guest or Public and turn off Network Discovery and File and printer sharing
  • Go to All Networks, check turn off Public folder sharing, Use 128, and turn on password protected sharing

Lock IP

  • Go to Control Panel
  • Go to Network and Sharing Center
  • Click Connections: Wi-Fi(xxx)
  • Click Details and make a snapshot to get: IPv4 Address, IPv4 Subnet Mask, and IPv4 Default Gateway, then close
  • Click Properties
  • Find and click Internet Protocol Version 4(TCP/IPv4) under Networking tab
  • Click Properties, check Use the following IP address and fill in IPv4 Address, IPv4 Subnet Mask, and IPv4 Default Gateway with information from step 4.
  • Click OK

Windows Media Player

  • Go to Stream and click turn on media streaming
  • Check Allow

Android TV

  • Install WX Player
